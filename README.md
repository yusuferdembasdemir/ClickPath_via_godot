
---------TÜRKÇE-----------------

Tıklama Mekaniği: Ekrana tıklayarak puan (Clicks) toplayın.

Market Sistemi: Kazandığınız puanlarla yeni arka planlar ve geliştirmeler satın alın.

Kayıt Sistemi: Oyunu kapatsanız bile puanlarınız ve satın aldığınız eşyalar güvende kalır (Config sistemi).

Teknik Detaylar
Oyun Motoru: Godot 4.6.1

Programlama Dili: GDScript

Node Yapısı:
Global.gd: Veri yönetimi ve kaydetme işlemleri (Autoload).

Main.gd: Ana oyun döngüsü ve UI güncellemeleri.

Shopscene.gd: Market etkileşimleri ve satın alma kontrolü.

-----------ENGLİSH------------

Clicking: Collect points (Clicks) by clicking on the screen.

Shop System: Purchase new backgrounds and upgrades using your earned points.

Save System: Your points and purchased items are securely stored via the Config system, even if you close the game.

Technical Details
Game Engine: Godot 4.6.1

Programming Language: GDScript

Node Structure
Global.gd: Data management and save operations (Autoload).

Main.gd: Core game loop and UI updates.

Shopscene.gd: Shop interactions and purchase logic.
